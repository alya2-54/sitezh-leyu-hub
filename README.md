# sitezh-leyu-hub

一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面的存储与分发集合。

## 项目简介

本项目旨在整理、归档各类独立 HTML 页面，便于统一管理和访问。  
每个页面均为自包含的 HTML 文件，不依赖外部服务或特定环境。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 归档页面目录
│   ├── page-1.html     # 示例页面
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/`：存放所有归档的独立 HTML 页面。
- `assets/`：存放页面所需的公共资源文件（如 CSS、JS、图片等）。

## 页面归档说明

- 每个页面应保持独立，尽量减少对外部资源的依赖。
- 页面文件命名建议使用英文小写，以连字符分隔，如 `example-page.html`。
- 若页面使用公共资源，请将资源文件放入 `assets/` 目录，并在页面中引用相对路径。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，或对现有页面进行改进。
- 请确保新增页面不包含恶意代码、侵犯版权内容或违反法律法规的信息。
- 本仓库为静态归档用途，不提供动态服务或用户交互功能。
- 管理员会定期检查页面可用性与内容合规性，并可能移除不符合要求的文件。
